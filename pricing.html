常見問題

可以隨時升級或降級方案嗎？

可以。Adforget 採用月訂閱制，您可以隨時在帳戶設定中調整方案，變更將於下一個計費週期生效。

免費版有使用期限嗎？

沒有。入門版永久免費，適合剛起步的個人賣家體驗核心功能。當業務成長時，可隨時升級至專業版或企業版。

什麼是 Agent 自動分析？

Agent 是 Adforget 內建的 AI 助手，分別負責商品頁優化、廣告分析、社群內容、營運風險控管。專業版起可自動判讀數據並給出建議。

年付方案可以退款嗎？

年付方案享有 2 個月免費優惠。購買後 14 天內可全額退款，14 天後依剩餘月份比例退款。

企業版有客製化功能嗎？

企業版包含專屬客戶成功經理，可根據您的業務需求客製報表、自動化流程與 API 串接。如需進一步客製開發，請聯繫銷售團隊。

加值服務可以單獨購買嗎？

可以。AI 素材生成、廣告代操、API 串接等加值服務均可單獨購買，無需綁定特定方案。
